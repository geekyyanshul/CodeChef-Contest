# CodeChef Contest Solutions

A collection of competitive programming solutions for various [CodeChef](https://www.codechef.com/) Starters contests, primarily written in Java with occasional Python.

## Repository Structure

Each folder corresponds to a CodeChef contest and contains sub-folders for individual problems:

| Contest | Problems Solved |
|---------|-----------------|
| [start215c](./start215c) | Problem 1, Problem 2, Problem 3 and so on|

## Languages Used

- **Java** – primary language for most solutions
- **Python** – used for select problems

## How to Run

### Java
```bash
javac SolutionFile.java
java SolutionFile
```

### Python
```bash
python3 solution.py
```
